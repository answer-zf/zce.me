## Getting Started

```shell
# clone repo
$ git clone https://github.com/zce/zce.me.git
$ cd zce.me
# install dependencies
$ yarn # or npm install
```

### Available Scripts

```shell
# serve with hot reload at http://localhost:8000
$ yarn dev

# format the source files.
$ yarn format

# build for production with minification
$ yarn build
```
